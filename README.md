# AnyLiquid

**AnyLiquid** is an open-source, modular trading infrastructure inspired by Hyperliquid — designed to make high-performance on-chain trading accessible, customizable, and fully transparent.

---

## 🚀 Overview

AnyLiquid aims to provide a **permissionless, extensible, and developer-friendly** framework for building decentralized perpetual exchanges and trading systems.

Our mission is simple:

> Bring institutional-grade trading performance to open, community-owned infrastructure.

---

## ✨ Key Features

* ⚡ **High-performance matching engine**
  Built for low-latency execution and scalability.

* 🔓 **Fully open-source**
  No black boxes — everything is transparent and auditable.

* 🧩 **Modular architecture**
  Easily plug in custom components (risk engine, matching logic, APIs).

* 🌐 **On-chain + off-chain hybrid design**
  Balance performance with trust minimization.

* 🛠 **Developer-first tooling**
  APIs, SDKs, and documentation designed for builders.

---


## 🧱 Architecture

AnyLiquid follows a hybrid architecture:

* **Off-chain components** handle:

  * Order matching
  * Sequencing
  * Real-time trading

* **On-chain components** handle:

  * Settlement
  * Asset custody
  * Verification

This design enables **CEX-like speed with DEX-level transparency**.
